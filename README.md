# 📊 Análisis de Datos y Dashboard Interactivo en Excel

📝 **Descripción del Proyecto**
Este proyecto trata la metodología completa de análisis de datos utilizando Excel. El objetivo principal es transformar datos brutos en elementos accionables, operando a través de tres fases:
1. **Importación y preparación de datos** (mediante Power Query).
2. **Análisis de datos** (utilizando Tablas Dinámicas y funciones analíticas).
3. **Visualización** (Creación de un dashboard interactivo simulando una aplicación).

## 📊 Dataset Utilizado
Para este proyecto, se ha trabajado con el dataset de Data Science for Good: Kiva Crowdfunding, el cual contiene información detallada sobre préstamos de otorgados por esta organización.

## 🛠️ Tecnologías Utilizadas
* **Microsoft Excel:** Herramienta de análisis y visualización.
* **Power Query:** Para la extracción, transformación y carga de los datos.
  
## 🎯 Competencias
Durante el desarrollo de este proyecto se han necesitado y puesto en práctica las siguientes competencias:
1. Configuración y automatización del entorno de trabajo (Power Query).
2. Evaluación y limpieza profunda de conjuntos de datos complejos.
3. Desarrollo de interfaces de usuario dinámicas, intuitivas y funcionales (Dashboard).
4. Gestión de la información.
